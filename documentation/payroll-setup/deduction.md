# Deduction Management

## Overview
Manage specific deductions from employee salaries, such as penalties, loan recoveries (manual), or other adjustments.

## Actions

### 1. Create Deduction
- **Description:** Apply a deduction to an employee's salary.
- **Fields:** Employee, Deduction Type (Head), Amount, Month/Year.
- **Path:** `/hr/payroll-setup/deduction/create`

### 2. View Deductions
- **Description:** List of all deductions applied.
- **Path:** `/hr/payroll-setup/deduction/view`
