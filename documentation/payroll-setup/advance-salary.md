# Advance Salary

## Overview
Allows employees to request a portion of their salary in advance, which is then deducted from the upcoming payroll.

## Actions

### 1. Create Request
- **Description:** Submit a request for advance salary.
- **Path:** `/hr/payroll-setup/advance-salary/create`

### 2. View Requests
- **Description:** Managers and HR can view, approve, or reject advance salary requests.
- **Path:** `/hr/payroll-setup/advance-salary/view`

### 3. Request Forwarding
- **Description:** Configure the approval chain for advance salary requests.
- **Path:** `/hr/request-forwarding?type=advance-salary`
