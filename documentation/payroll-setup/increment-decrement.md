# Increment / Decrement

## Overview
Handles changes to employee base salaries.

## Actions

### 1. Create
- **Description:** Record a salary increment (raise) or decrement.
- **Fields:** Employee, New Salary Amount, Effective Date, Reason.
- **Path:** `/hr/payroll-setup/increment/create`

### 2. View
- **Description:** History of all salary changes for audit and tracking purposes.
- **Path:** `/hr/payroll-setup/increment/view`
