# Loan Requests

## Overview
Manages long-term company loans provided to employees, including repayment schedules and installments.

## Actions

### 1. Create Request
- **Description:** Apply for a loan.
- **Fields:** Loan Amount, Reason, Repayment Period (Months).
- **Path:** `/hr/loan-requests/create`

### 2. View & Reports
- **Description:** Track loan status, outstanding balances, and repayment history.
- **Path:** `/hr/loan-requests/view`

### 3. Request Forwarding
- **Description:** Set up the approval workflow for loan applications.
- **Path:** `/hr/request-forwarding?type=loan`
