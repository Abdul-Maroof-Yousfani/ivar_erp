# Allowance Management

## Overview
Manage specific allowances assigned to employees. While general allowances might be part of the salary structure, this module handles specific or ad-hoc allowances.

## Actions

### 1. Create Allowance
- **Description:** Assign an allowance to an employee.
- **Fields:** Employee, Allowance Type (Head), Amount, Effective Date.
- **Path:** `/hr/payroll-setup/allowance/create`

### 2. View Allowances
- **Description:** List of all active and past allowances assigned.
- **Path:** `/hr/payroll-setup/allowance/view`

### 3. Bank Report (Allowance)
- **Description:** Specific bank transfer reports for allowance disbursements if paid separately.
- **Path:** `/hr/payroll-setup/allowance/bank-report`

### 4. Allowance Payslip
- **Description:** Generate slips specifically for allowance payments.
- **Path:** `/hr/payroll-setup/allowance/payslip`
