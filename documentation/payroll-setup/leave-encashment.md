# Leave Encashment

## Overview
Allows employees to exchange their accumulated unused leaves for monetary compensation.

## Actions

### 1. Create
- **Description:** Process a leave encashment request.
- **Path:** `/hr/payroll-setup/leave-encashment/create`

### 2. List
- **Description:** View history of encashments.
- **Path:** `/hr/payroll-setup/leave-encashment/list`

### 3. Request Forwarding
- **Description:** Approval workflow for encashment requests.
- **Path:** `/hr/request-forwarding?type=leave-encashment`
