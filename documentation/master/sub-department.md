# Sub Department Master

## Overview
Sub-departments are subdivisions within a main Department (e.g., "Development" under "IT").

## Actions
- **Add:** Create a new sub-department linked to a main department.
  - *Path:* `/master/sub-department/add`
- **View:** List all sub-departments.
  - *Path:* `/master/sub-department/list`

## Data Fields
- **Name:** The name of the sub-department.
- **Department:** The parent department it belongs to.
