# Salary Breakup Master

## Overview
Defines the components of a salary structure (e.g., Basic Salary 60%, HRA 20%, Utility 10%).

## Actions
- **Add:** Create a new breakup template.
  - *Path:* `/master/salary-breakup/add`
- **View:** List breakup templates.
  - *Path:* `/master/salary-breakup/list`
