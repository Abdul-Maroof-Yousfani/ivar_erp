# Department Master

## Overview
Departments represent the high-level functional divisions within the organization (e.g., IT, HR, Finance).

## Actions
- **Add:** Create a new department.
  - *Path:* `/master/department/add`
- **View:** List all departments, edit or delete existing ones.
  - *Path:* `/master/department/list`

## Data Fields
- **Name:** The name of the department.
- **Status:** Active/Inactive.
