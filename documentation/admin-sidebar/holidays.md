# Holidays

## Overview
Manage the organization's holiday calendar. This includes public holidays, company-specific holidays, and weekends. These dates are used when calculating attendance and leaves.

## Actions

### 1. Create (Add Holiday)
- **Description:** Add a new holiday to the calendar.
- **Fields:** 
  - Holiday Name (e.g., "Independence Day")
  - Date(s)
  - Type (Public, Optional, etc.)
- **Path:** `/hr/holidays/add`

### 2. List
- **Description:** View the list of all upcoming and past holidays configured in the system.
- **Path:** `/hr/holidays/list`
