# Leaves Setup

## Overview
Handles leave management, including application, approval workflows, and leave balances.

## Sections

### 1. Create Leave
- **Description:** Apply for leave on behalf of an employee or for oneself.
- **Path:** `/hr/leaves/create-leaves`

### 2. View Requests
- **Description:** Dashboard for managers/HR to view, approve, or reject leave applications.
- **Path:** `/hr/leaves/requests`

## Key Features
- **Leave Types:** Integration with Master Setup (Sick, Casual, Annual, etc.).
- **Balance Tracking:** Automatic deduction from leave balances.
- **Approval Flow:** Supports multi-level approval if configured.
