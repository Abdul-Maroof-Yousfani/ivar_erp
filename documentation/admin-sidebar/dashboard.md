# Dashboard

## Overview
The Dashboard is the landing page for the HR Management System. It provides a high-level overview of the organization's key metrics and quick access to important functions.

## Features
- **Key Metrics:** View total employees, departments, active/inactive staff.
- **Attendance Overview:** Quick glance at today's attendance stats.
- **Recent Activities:** Log of recent system actions.
- **Quick Links:** Shortcuts to frequently used modules.

## Access
- **Path:** `/hr`
- **Permissions:** Accessible to all authenticated users (Permissions may vary for specific widgets).
